Installation Guide

STEP 1

Download & Install

Download the Print Agent application for your operating system (Windows or Linux). Install and open the application. You'll see a clean interface ready for setup.

STEP 2

Authorize Your PC

Click "Step 1: Authorize PC". Your browser will show a security warning about an insecure HTTP connection. This is normal - accept it to authorize communication between Odoo and the local print agent.

STEP 3

Confirm Activation

After authorization, you'll see: "Printing service is active and authorized". The agent is now running and ready to receive print jobs from Odoo.

STEP 4

Configure in Odoo

In Odoo, go to Settings and find "Odoo Direct Print ‚Äì MateusC7". In Print Agent Configuration, copy the IP from the agent's "Step 2" button (e.g., de>http://192.168.1.10:5000) and paste it here.

STEP 5

Assign Printers to Reports

Navigate to Report Management. Select any report (e.g., Invoice PDF) and enter the exact printer name in Agent Printer Name. Copy it from the agent's printer list for accuracy.

STEP 6

Configure POS (Optional)

For Point of Sale, open POS settings ‚Üí Connected Devices ‚Üí Local Printing (Agent). Select your receipt printer. For kitchen printers, go to Preparation ‚Üí Manage Printers and assign printers by category.

STEP 7

Kitchen Printers Setup

In the Agent Printer Name field, type the exact name of the kitchen printer you want to use. This allows you to print different orders by category - ideal for restaurants, bars, or businesses with multiple stations.
